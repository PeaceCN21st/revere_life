## 布衣之怒

秦王怫然怒，谓唐雎曰：“公亦尝闻天子之怒乎？”

唐雎对曰：“臣未尝闻也。”

秦王曰：“天子之怒，伏尸百万，流血千里。”

唐雎曰：“大王尝闻布衣之怒乎？”

秦王曰：“布衣之怒，亦免冠徒跣，以头抢地耳。”

唐雎曰：
> ”此庸夫之怒也，非士之怒也。
>
> 夫专诸之刺王僚也，彗星袭月
>
> 聂政之刺韩傀也，白虹贯日；要离之刺庆忌也，仓鹰击于殿上。
>
> 此三子者，皆布衣之士也，怀怒未发，休祲(jìn)降于天，与臣而将四矣。
>
> 若士必怒，伏尸二人，流血五步，天下缟素，今日是也。“

挺剑而起。
 
 
 
 
 
- 来源：
> 《战国策·魏策四》中的故事“唐雎不辱使命”
